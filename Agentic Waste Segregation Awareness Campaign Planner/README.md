## 📌 Project Overview

This project is an **Agentic AI-based Waste Segregation Awareness Campaign Planner** built using **Langflow**.

It intelligently designs, executes, and improves sustainability awareness campaigns based on:
- 👥 Audience type  
- 📍 Location  
- 💰 Budget  
- 🔄 Feedback  
- ♻️ 5R Sustainability model  

---

## 🌍 Real-World Problem

Waste awareness campaigns fail due to:
❌ Generic messaging  
❌ No audience targeting  
❌ No feedback learning loop  
❌ Poor engagement strategies  

---

## 🤖 Proposed Solution (Agentic AI System)

This system acts as an intelligent AI Agent that:

✔ Understands user goal  
✔ Plans campaign automatically  
✔ Generates posters & content  
✔ Creates quizzes & activities  
✔ Analyzes feedback  
✔ Improves future campaigns using memory  

---

## 🧠 System Architecture

```
👤 User Input
      ↓
🤖 Main Agent (Langflow)
      ↓
────────────────────────────────
🧩 AI Tools Layer:
• Audience Profiler

• Campaign Planner
• Poster Generator 🪧
• Social Media Generator 📱

────────────────────────────────
      ↓
🧠 Memory (Long-term Storage)
      ↓
📤 Final Campaign Output
```

## 🧩 Langflow Components Used

### 🔷 Core Components
- Chat Input 💬
- Chat Output 📤
- Agent (Main Brain) 🧠

### 🔷 Tool Components (Prompt Templates - TOOL MODE ON ✅)

✔ Audience Profiler → Identifies audience behavior  
✔ 5R Strategy Tool → Selects sustainability methods  
✔ Campaign Planner → Builds structured campaign plan  
✔ Poster Generator → Creates poster text + prompts  
✔ Social Media Generator → Creates posts & hashtags  
✔ Quiz Generator → Builds MCQs  
✔ Feedback Analyzer → Processes user feedback  
✔ Reflection Tool → Improves campaign strategy  
✔ Report Generator → Final structured report  

---

### 🔷 Custom Component ⚙️

✔ Impact Score Calculator (Python Logic)

```
📊 Impact Score Formula:
impact_score = (feedback_score + awareness_score) / 2
```

Used for measuring campaign effectiveness.

---

### 🔷 Memory Component 🧠

Stores:
- Past campaigns  
- Feedback history  
- Improvements  
- User preferences  

Enables **learning over time (Agentic behavior)**

## 🔄 Agent Workflow

### Step 1: User Input
```
Create a 7-day waste awareness campaign for engineering students in Bengaluru. Budget ₹5000.
```

---

### Step 2: Agent Calls Tools

🤖 Automatically triggers:
- Audience Profiler  
- 5R Strategy Selector  
- Campaign Planner  
- Poster Generator  
- Quiz Generator  
- Social Media Generator  

---

### Step 3: Feedback Loop 🔁

User Feedback:
```
Students liked quizzes but ignored posters.
```

Agent uses:
- Feedback Analyzer  
- Reflection Tool  
- Memory  

---

### Step 4: Improvement Cycle

✔ Adds interactive posters  
✔ Reduces static content  
✔ Improves engagement strategy
